# 🌆 Smart City Assistant

The Smart City Assistant is an AI-powered web application that helps urban planners and citizens make sustainable decisions using data-driven insights. It integrates multiple features including eco-tip generation, policy summarization, KPI forecasting, anomaly detection, and sustainability report generation.

## 🔧 Tech Stack

- **Frontend**: Streamlit  
- **Backend**: FastAPI  
- **LLM**: IBM Watsonx (Granite Model)  
- **Data**: CSV, TXT input handling  
- **PDF Generation**: FPDF

## 🚀 Key Features

- 🌿 Generate eco-friendly tips on topics like water, waste, traffic  
- 📊 Forecast KPIs using uploaded datasets  
- 📘 Generate professional sustainability reports (Markdown & PDF)  
- 📄 Summarize lengthy policy documents for easier understanding  
- 🔍 Detect anomalies in city metrics with threshold logic  
- 💬 Chat with an AI assistant using optional context  
- 📋 View and manage citizen feedback logs
